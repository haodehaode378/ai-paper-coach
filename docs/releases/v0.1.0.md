# AI Paper Coach v0.1.0

发布日期：2026-04-15

## 版本概览
`v0.1.0` 是首个可分享版本，覆盖论文导入、结构化分析、七问评审、整理输出与报告导出的完整闭环。

## 新增与改进
- 完整流程：`ingest -> analyze -> review -> finalize -> report`
- 七问结构化输出与长度阈值校验（摘要 `>=800`，复现指导 `>=1000`，七问单题 `>=700`）
- 历史记录与已保存报告管理（包含删除）
- `finalize` 可观测增强：记录每次模型调用耗时、是否触发 repair、重试次数（trace 可回放）
- 标题提取增强：补充 arXiv API 标题兜底、修复占位标题识别问题
- 复现指导空内容修复：增加 finalize/review 合并兜底逻辑
- 前端运行状态增强：阶段进度、预计剩余时间、当前模型展示
- 安全基础能力：CORS 白名单、可选 API Key 鉴权、上传大小限制、统一响应 envelope

## 安全说明
- `.env` 及变体已忽略，不入仓库
- 支持 `gitleaks` 扫描；当前 git 历史扫描结果为 `0` 命中

## 已知问题
- 复杂论文在 `finalize` 阶段仍可能出现尾段耗时偏高
- 极少数 PDF 元数据缺失且无法匹配 arXiv 时，标题仍可能降级为占位文本

## 升级建议
- 拉取最新代码后，执行一次完整流程重跑旧任务，以应用标题与复现指导修复
- 若对外部署，建议启用 `APC_REQUIRE_API_KEY=1` 并配置 `APC_ALLOWED_ORIGINS`
